Work Day Scheduler

A user friendly application for planning out your day
1. Type any name you would like to store as an event in the according hour slot.
2. Click the lavender button with the save icon to record your event.
3. Voila! Your event is now recorded and you can visit this page again to view your events through your busy day!

Green = Future Event | Yellow = Present Event |Grey = Past Events
